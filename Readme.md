# Thief Detecter
When your treasure is being taken away, the warning lights and electric horn will let you know.<br>
Also making a notifition via Gmail with esp32.

## Components and supplies
1. Arduino UNO
2. ESP 32
3. Relay
4. Warning Litghts & Electirc Horns
5. Force Sensor
6. Resistor 3.3k ohm
7. Jumper wires
